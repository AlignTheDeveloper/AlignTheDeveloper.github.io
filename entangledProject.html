Entangled

(Interactive Comic)

This is a Web comic / app that I worked on with other developers that aims to explain how quantum entanglement works. I learned a lot about how to condense big and convoluted ideas into easily understood allegories in this project. I wrote the script for the comic, and we as a team brainstormed ways to further visualize entanglement. My biggest contribution to this visualization was to create a matching game that populates a grid of prefabs randomly every time it is loaded. The project was fun to build because I really liked the team I was working with and the ideas we had to showcase this thing.
